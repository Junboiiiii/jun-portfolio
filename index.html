🏆 Final Year Project
2026
AI-Lips-Aesthetic-Recommender
Asia Pacific University · Capstone Project
AI Lips Aesthetic Recommender is an AI-driven system designed to assist in aesthetic pre-consultation by providing personalized lip enhancement recommendations. The problem it addresses is the lack of objective, data-driven guidance in aesthetic planning, which is often subjective and depends heavily on manual consultation. My approach was to combine machine learning with facial feature analysis. I extracted geometric and proportional features from facial data, then used Random Forest for region classification and K-Nearest Neighbors (KNN) for similarity-based recommendation. This allowed the system to generate tailored suggestions based on users with similar facial characteristics. The system was built using Python, with Scikit-learn, Pandas, and NumPy for data processing and model development, and integrated into an end-to-end pipeline for real-time inference. The project achieved strong and consistent classification performance and was awarded a 4.0 grade, demonstrating both technical effectiveness and practical applicability in real-world scenarios.
